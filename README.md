# Lab6-web

para correr es go run main.go

Objetivos:

Que los alumnos practiquen conceptos básicos de javascript.

Que los alumnos aprendan a modificar el dom.

Requerimientos:

Para optar a una nota distinta de 0, deben realizar un chat que se conecte al servidor realizado en la clase. El chat debe estar publicado en internet. No deben utilizar librerías externas (de javascript) para éste ejercicio, pero pueden utilizar cualquier función de javascript que corra en navegadores modernos.

SE HIZO LOS SIGUIENTES REQUERIMIENTOS
Puntos:

(50 puntos) [Criterio subjetivo] Según qué tan interesante es el diseño de su interfaz
(10 puntos) Por colocar el campo de texto para escribir el mensaje en la parte inferior de la pantalla (no debe ocultarse cuando hay muchos mensajes)
(10 puntos) Por limitar la cantidad de caracteres permitidos a 140 en cada mensaje 
(10 puntos) Por agregar hacer "submit" del mensaje utilizando "Enter" 
(10 puntos) Auto-refresh de la lista de mensajes
(20 puntos) Por preservar el scroll después de recibir nuevos mensajes
